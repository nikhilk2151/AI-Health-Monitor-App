# AI Health Mentor – Prototype

AI Health Mentor is a modern AI-powered wellness platform designed to help users improve their **physical fitness, mental health, and lifestyle habits** using Artificial Intelligence.

This repository contains the **frontend prototype (Home Page UI)** created for a hackathon project.  
Currently, only the **homepage design** is implemented. Backend and AI features will be added in future development stages.

---

## Project Idea

Many people struggle with maintaining healthy habits due to busy lifestyles, stress, and lack of personalized guidance. Existing apps usually focus on only fitness or mental health, but not both together.

AI Health Mentor aims to solve this problem by providing:

- AI-based health insights
- personalized fitness recommendations
- mental wellness guidance
- yoga and meditation support
- progress tracking dashboard
- nearby healthcare clinic suggestions
